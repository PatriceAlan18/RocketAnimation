A Java 2D animation made with my own graphics methods

All of the code is in Spanish.
The main method is in the package "proyecto", in the class "Principal".

The animation features a rocket traveling through space.
It includes a random number of stars, which twinkle every second.
The background is created with a random color pattern.
The animation also has several animated elements,such as meteors and planets, with their values as random parameters.
Additional elements include a rocket, an astronaut, and some musical notes as well.
